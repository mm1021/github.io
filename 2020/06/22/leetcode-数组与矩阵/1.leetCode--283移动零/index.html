1. LeetCode–283移动零

题目连接：https://leetcode-cn.com/problems/move-zeroes/

题目类型：简单数组问题

1.1 问题描述

  1. 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
  2. 示例：
    • 输入: [0,1,0,3,12]
    • 输出: [1,3,12,0,0]
  3. 说明：
    • 必须在原数组上操作，不能拷贝额外的数组。
    • 尽量减少操作次数。

1.2 问题分析

  1. 注意关键点：不能拷贝额外的数组和保持非零元素的相对顺序
  2. 所以先遍历一次数组把非零数往前放，第一次遍历后的示例数组为1,3,12,3,12 通过示例这个我们发现会非零数组的值按原来的顺序排列，并且此时的index=3正好指向第二个 3，所以我们后面只需要把index及以后的数全置为0就可以了。

1.3 代码实现

/**
 * 	细节： nums[index] = nums[i]; index++;
 * 	可以写成 nums[index++] = nums[i];
 * @param nums 需要处理的数组
 */
public static void moveZeroes(int[] nums) {
	
	// 设置下标
	int index = 0;
	
	for (int i = 0; i < nums.length; i++) {
		if(nums[i]!=0) {
			nums[index] = nums[i];
			index++;
		}
	}

	while(index < nums.length) {
		nums[index] = 0;
		index++;
	}
	
	// 测试代码
	System.out.println(index);
	for (int i = 0; i < nums.length; i++) {
		System.out.println(nums[i]);
	}

1.4 测试代码

public class MoveZeroes {
	public static void main(String[] args) {
		int[] nums = {0,1,0,3,12};
		moveZeroes(nums);
	}
}
