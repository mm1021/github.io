1. leetCode–217.存在重复元素

  • 题目连接：https://leetcode-cn.com/problems/contains-duplicate/
  • 题目类型： 简单数组

1.1 问题描述

  1. 给定一个整数数组，判断是否存在重复元素。如果任意一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。

  2. 示例 1:

    输入: [1,2,3,1]
输出: true

  3. 示例 2:

    输入: [1,2,3,4]
输出: false

  4. 示例 3:

    输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

1.2 问题分析

解法一：
  1. 此问题是找数组中是否有重复的元素，我们可以通过HashMap的key来储存数组的值，value来储存该数组的下标，如果源数组中存在重复的元素则会有以下条件map.size() != arr.length。
  2. 注意细节问题，new HashMap<>(nums.length);如果不添加这个时间会慢一秒。
解法二：
  1. 我们可以先把数组通过Arrays.sort(arr)进行排序，然后遍历数组判断相邻的两个元素是否相等，如果相等返回为true，否则为false。
  2. Arrays.sort(arr);排序算法是由Vladimir Yaroslavskiy，Jon Bentley和Joshua Bloch提供的双轴快速排序。 该算法在许多数据集上提供O（n log（n））性能，导致其他快速排序降级为二次性能，并且通常比传统（单轴）Quicksort实现更快。

1.3 代码实现

解法一：
public static boolean containsDuplicate(int[] nums) {

    // 创建map集合
    Map<Integer,Integer> map = new HashMap<>(nums.length);

    // key来储存数组的值，value来储存该数组的下标
    for (int i = 0; i < nums.length; i++) {
        map.put(nums[i],i);
     }
    
    // 如果map.size()==nums.length说明源数组没有重复元素
    if(map.size()==nums.length){
        return false;
     }
    return true;
}
解法二：
public boolean containsDuplicate(int[] nums) {
    Arrays.sort(nums);
    for (int i = 0; i < nums.length - 1; ++i) {
        if (nums[i] == nums[i + 1]) {
            return true;
        }
    }
    return false;
 }
